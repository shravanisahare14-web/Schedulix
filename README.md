# ⚙️ Schedulix

### Constraint-Aware Scheduling & Resource Optimization Platform

Schedulix is an scheduling and resource optimization platform designed to solve complex workforce planning, task allocation, dependency management, and capacity optimization problems.

The platform combines Graph Theory, Priority Queue Scheduling, Constraint Programming, and Google OR-Tools CP-SAT Optimization to generate intelligent schedules while maximizing operational efficiency and business value.

---

## 🚀 Key Highlights

- Dynamic Task Configuration Engine
- Dependency-Aware Scheduling
- Greedy Priority Queue Baseline Scheduler
- Google OR-Tools CP-SAT Optimizer
- Resource Constraint Simulation
- Workload Scaling Analysis
- Interactive Gantt Timeline Visualization
- Real-Time What-If Analysis
- CSV Export Engine
- Audit Report Generator
- Enterprise-Style Operations Dashboard

# 🧠 Algorithms Used

## Greedy Priority Queue Scheduler

Uses:

- Earliest Due Date (EDD)
- Min Heap
- Dependency Resolution
- Resource Availability Tracking

Complexity:

```text
O(V log V + E)
```

---

## CP-SAT Optimization Engine

Uses:

- Constraint Programming
- Integer Optimization
- No-Overlap Constraints
- Dependency Constraints
- Objective Maximization

Powered by:

```text
Google OR-Tools
```

# 🖥 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Schedulix.git
```

Move into project directory:

```bash
cd Schedulix
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
